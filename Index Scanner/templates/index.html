HeuristiX Guide

What does this tool do?

HeuristiX crawls a website and detects malware, malicious scripts, phishing pages, brand impersonation, suspicious links, obfuscated code, and credential harvesting forms. It is not limited to phishing — it finds any kind of threat on a site.

Risk Score (0–100)

  • Green (71–100) — Site is safe. No or very few threats found.
  • Yellow (51–70) — Moderate risk. Some suspicious indicators detected.
  • Red (0–50) — Dangerous. Many threats found. Do not trust this site.

Threat Severity Levels

  • CRITICAL — Known malware or active exploitation. Immediate danger.
  • HIGH — Strong phishing, brand impersonation, or credential harvesting.
  • MEDIUM — Suspicious patterns, unusual TLDs, or multiple scam keywords.
  • LOW — Minor indicators, low-confidence signals.

Discovered Files

Every file the crawler finds (HTML pages, external scripts, inline scripts) is listed with a Download button so you can inspect the raw source code yourself.

Discovered Links

All links found on the crawled pages are listed. You can click any link to open it in a new tab and verify it manually.

Source Types

  • page — An HTML page crawled from the target site.
  • external_script — A JavaScript file loaded from an external URL.
  • inline — Inline JavaScript embedded directly in a page.
